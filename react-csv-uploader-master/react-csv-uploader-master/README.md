# react-csv-uploader

It is a simple example of react-saga using [create react app](https://github.com/facebookincubator/create-react-app)

clone it and run:

```
  yarn
  yarn start
```

Then open http://localhost:3000/ to see your app.

Demo csv format is

| firstname | lastname |
| --------- | -------- |
| Bobby     | Thomas   |
| Jason     | Wright   |
| Santiago  | Norris   |
